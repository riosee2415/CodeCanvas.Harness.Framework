## 작성 규칙
- 각 ADR은 **하나의 결정**만 다룬다. 번호는 부여 후 변경하지 않는다.
- `Status`: `제안됨` | `채택됨` | `폐기됨` | `대체됨(ADR-NNN으로)`. 결정을 뒤집을 때는 기존 ADR을 지우지 말고 `폐기됨`으로 바꾸고 새 ADR을 추가한다 (결정의 역사를 보존).
- `대안`: 검토했으나 선택하지 않은 안과 그 탈락 이유를 적는다. 이게 없으면 같은 논의가 반복된다.
